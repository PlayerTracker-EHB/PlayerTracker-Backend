## **Prerequisites**

Before you begin, ensure you have met the following requirements:

•Python 3.x installed on your machine.

•Pipenv installed. You can install it using pip:
```bash
pip install pipenv
```
## **Getting Started**

Follow these instructions to set up the project on your local machine after cloning it.

## **Cloning the Repository**

1.Open your terminal (Command Prompt on Windows or Terminal on macOS).

2.Clone the repository using Git:
```bash
git clone https://github.com/PlayerTracker-EHB/PlayerTracker-Backend.git
```
3.Navigate into the project directory:
```bash
cd PlayerTracker-Backend
```
## **Setting Up the Environment**

### **On Windows**

1.**Open Command Prompt**:

▪Press Win + R, type cmd, and hit Enter.

2.**Install Dependencies**:

Run the following command to install the required packages:
```bash
pipenv install
```
3.**Activate the Pipenv Shell**:

To enter the virtual environment, run:
```bash
pipenv shell
```
4.**Run Your Application**:

Now you can run your application or scripts. 

5.**Exit the Environment**:

When done, type:
```bash
exit
```
or press **Ctrl + D** to deactivate the Pipenv shell.

### **On macOS**

1.**Open Terminal**:

▪You can find it in Applications > Utilities > Terminal.

2.**Install Dependencies**:
